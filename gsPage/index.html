API

初始化 : $("#warp").gsPage();

上一页 : $.fn.gsPage.prev();

下一页 : $.fn.gsPage.next();

绑定方向键 : $.fn.gsPage.keyDownFn(); 也可以初始化时设置$("#warp").gsPage({"keydown":true});

绑定滑动事件 : $.fn.gsPage.swipeFn(); 也可以初始化时设置$("#warp").gsPage({"swipe":true}); ---依赖组件jquery.touchSwipe

传入自定义隐藏动画 : $.fn.gsPage.defaults.outAnimate = "xxx";

传入自定义隐藏动画 : $.fn.gsPage.defaults.comeAnimate = "xxx";

传入动画速度 : $.fn.gsPage.defaults.animateSpeed = 555;

动画效果参数参考 Animate.css,取消自定义设置为false即可
